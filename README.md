# 📖 Descripción del Proyecto

ConectaComercio es una plataforma web desarrollada bajo una arquitectura cliente-servidor que permite la interacción entre **mayoristas y tenderos**, mediante servicios web REST.

El sistema permite la gestión de usuarios, productos y pedidos, aplicando autenticación, control de roles y persistencia de datos en base de datos relacional.

---

# 🎯 Objetivo General

Diseñar e implementar servicios web REST que permitan la gestión de usuarios, productos y pedidos dentro de una plataforma comercial con control de acceso por roles.

---

# 🎯 Objetivos Específicos

- Implementar autenticación de usuarios.
- Gestionar roles de acceso (admin, mayorista, tendero).
- Desarrollar CRUD de usuarios.
- Gestionar productos publicados por mayoristas.
- Permitir consulta de productos por tenderos.
- Implementar carrito de compras.
- Crear y almacenar pedidos.
- Aplicar seguridad en rutas.
- Integrar frontend y backend mediante API REST.
- Usar control de versiones con Git y GitHub.
